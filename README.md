## 概念
```
appId：应用的AppId，理解为项目唯一编号、一个git仓库等
clusterName：集群名，默认default，如果在开发者环境可以把一个开发者一个集群名
namespaceName：命名空间，默认application，在dev环境可以把一个分支作为一个命名空间
```
